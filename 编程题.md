1. 读取键盘上的输入T和余下T行数据, 计算出至少要经过几次运算可以使得运算次数最少? (腾讯笔试)

   输入:

   ```bash
   1
   10
   ```

   输出:

   ```bash
   4
   ```

   答:

   ```js
   //核心代码checker
   function checker(){
       let counter = 1;	//计算次数至少为1
       while(n!==1){		//取余结果为0\1\2, 对着三种状况分别判断
           if(n%3===1){
               n--;
               counter++;
           }
           if(n%3===2){
               n -= 2;
               counter += 2;
           }
           n = n/3;	//规律: 只有为3的倍数时次数最少
       }
       return counter;	//返回结果
   }
   ```
